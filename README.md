## 🚀 Getting Started

### Prerequisites
- Node.js 18 or higher
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Kingdom-Chronicles/efficiency-edge-site.git
```

2. Install dependencies:
```bash
cd efficiency-edge-site
npm install
```

3. Create a `.env` file based on `.env.example`:
```bash
cp .env.example .env
```

4. Start the development server:
```bash
npm start
```
